# OPEN_ASSUMPTIONS

- Expansion AI, marker and notification APIs must be checked against the exact installed target version before enabling release behavior.
- Stage objectives currently allow a timed fallback path; real terminal, generator and containment interactions still need a compile-tested action pass.
- Retry UI is a minimal RPC helper until the final layout and widget integration are added.
- Safe persistence currently saves JSON directly; atomic temp-save and last-known-good recovery should be added before release.
