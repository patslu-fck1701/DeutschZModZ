# DeutschZ_KotHZ Config Guide

## Pfad

`$profile:DeutschZ/KotHZ/Config/`

## Wichtige Bereiche

- Event aktivieren/deaktivieren
- Startzeit und Mindestspieler
- Zone, Radius, FlagpolePosition
- Marker und HUD
- CaptureTimeSeconds
- Zombies/Waves
- Rewards und Reward-Crate
- Cleanup
- Musik/Smoke/Fog

## Sichere Testwerte

- EnableKOTH = 1
- InitialDelaySeconds / Start nach Restart: 60 Sekunden
- EnableExpansionMarker = 1
- EnableExpansion3DMarker nach Serverziel testen
- EnableEventMusic = 0
- Smoke.EnableSmoke = 0
- Smoke.UseObjectSmoke = 0
- Smoke.UseParticleSmoke = 0
- Smoke.EnableFogHazard = 0
- Smoke.EnableSmokeCannons = 0

## Empfohlene Livewerte

- CaptureTimeSeconds hoeher setzen als im Test
- ZombieCounts langsam anheben
- RewardPools serverwirtschaftlich pruefen
- Marker klar sichtbar lassen

## Nicht blind aktivieren

- Musik
- Object-Smoke / Smoke-Grenades
- Particle-Smoke
- Fog-Hazard
- Smoke-Cannons
- RuntimeFlagpole-Attachment-Ketten

## Hinweise

Spieler muessen nur zur Signalstation laufen und den Radius halten. Keine Actiontaste noetig.
