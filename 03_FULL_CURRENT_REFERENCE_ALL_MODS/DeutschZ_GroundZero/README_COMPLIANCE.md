# README_COMPLIANCE

DeutschZ_GroundZero is an independent DeutschZ source module owned by Patrick Sluzalek / fck1701.

Allowed references are public DayZ, CF and Expansion API concepts only. No foreign mod files, class bodies, configs, assets, UI layouts, comments, headers or license text are copied into this module.

Expansion AI and marker integration is intentionally kept behind GroundZero adapter classes until the real target API names are verified against the installed Expansion version.
