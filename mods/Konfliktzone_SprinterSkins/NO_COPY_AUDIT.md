# NO_COPY_AUDIT - Konfliktzone_SprinterSkins

This addon is a clean retexture/config addon.

## Included

- Own Konfliktzone PAA decal files.
- Own `config.cpp`.
- Own `mod.cpp`.
- Own documentation.

## Not Included

- No MBM P3D files.
- No MBM scripts.
- No MBM sounds.
- No MBM RVMAT files.
- No MBM textures.
- No MBM PBO contents.
- No copied MBM class bodies.

## Referenced

- `MBM_Sprinter` as required addon.
- `MBM_Sprinter_BlackCargo` as inherited base class.
- MBM door base classes as inherited base classes.
- `MBM_MercedesSprinter\data\body_red.paa` as external texture reference.
