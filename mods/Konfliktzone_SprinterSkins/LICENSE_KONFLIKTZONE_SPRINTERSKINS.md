# 7. Technische Grundlage HiddenSelections

Die bekannte technische Anweisung von NamantH [DAYZ] an Einarvargr / Konfliktzone lautet sinngemaess:

```cpp
hiddenSelections[] = {"camo","camo_rear_glass"};
hiddenSelectionsTextures[] = {"SprinterPoliceLights\\data\\sprinterNYPD.paa","MBM_MercedesSprinter\\data\\black_transp.paa"};
```

Fuer Konfliktzone_SprinterSkins muss der erste Pfad durch den eigenen Pfad der Retexture-Mod ersetzt werden, zum Beispiel:

```cpp
hiddenSelectionsTextures[] = {"Konfliktzone_SprinterSkins\\data\\textures\\sprinter_konfliktzone_standard.paa","MBM_MercedesSprinter\\data\\black_transp.paa"};
```

Diese technische Information ist eine Anleitung zur erlaubten Retexture-Erweiterung. Sie ist keine Erlaubnis, MBM-Dateien, Modelle, Texturen oder PBO-Inhalte zu kopieren.
