# DeutschZ ServerConfig

Prepared local server configuration examples for DeutschZ.

Server: 193.135.10.126:20076

No secrets are stored here.

## Contents

- `economy/`: drop-in `cfgeconomycore.xml` / `economycore.xml` examples.
- `types/`: SNAFU, Modular Vest System, and DeutschZ special-item type definitions.
- `spawnabletypes/`: SNAFU and MVS spawnable attachments.
- `market/`: Expansion Market category files for SNAFU, MVS, and DeutschZ event-only special items.
- `trader/`: example Expansion trader category wiring.
- `profiles/DeutschZ/`: generated/default DeutschZ profile configs.
- `docs/`: setup notes, class discovery, replacement plans, and local test report.
