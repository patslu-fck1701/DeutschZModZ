# DeutschZ ConvoyZ Config-Beschreibung

Server-autoritatives Convoy-/Blackbox-Event mit Hack-Aktion, Gegnerwellen, Reward-Case, Cleanup und Fail-Safe-Verhalten.

## Abhaengigkeiten

Abhaengigkeiten: DeutschZ_Core, DeutschZ_ExpansionBridge.

## Profilepfade

- `$profile:DeutschZ/`
- Event-spezifisch: `$profile:DeutschZ/ConvoyZ/` beziehungsweise vorhandene Mod-Konfigpfade.

## Types.xml

types.xml ist nur als Template mit nominal=0 enthalten. Event-Items bleiben Event-/Trader-/Admin-Loot, kein normaler Mapspawn.
