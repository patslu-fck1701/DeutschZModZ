[h1]DEUTSCHZ CORE[/h1]
[i]The technical foundation of the DeutschZ mod system.[/i]

[h2]Description[/h2]
[list]
[*]Shared server profile paths and logging foundation
[*]Provider APIs for markers, messages and optional AI integrations
[*]Stable base for KotHZ, ConvoyZ, GroundZero and CourierZ
[*]Contains no own map-spawn economy items
[/list]

[h1]DEUTSCHZ CORE[/h1]
[i]Техническая основа модульной системы DeutschZ.[/i]

[h2]Описание[/h2]
[list]
[*]Общие пути профиля сервера и базовое логирование
[*]Provider API для маркеров, сообщений и опциональных AI-интеграций
[*]Стабильная основа для KotHZ, ConvoyZ, GroundZero и CourierZ
[*]Не содержит собственных предметов для случайного спавна на карте
[/list]
