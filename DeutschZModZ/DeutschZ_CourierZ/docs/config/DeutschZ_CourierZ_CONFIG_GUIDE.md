# DeutschZ_CourierZ Config Guide

## Pfad

`$profile:DeutschZ/CourierZ/`

## Wichtige Bereiche

- Enabled
- Scheduling / StartDelayMinSeconds / StartDelayMaxSeconds
- Pickup und DeliveryPosition
- Markers / CarrierMarker
- Notifications
- CaptureHoldSeconds
- Enemy/Wave Settings
- RewardOnDelivery
- CleanupDroppedCaseAfterSeconds

## Sichere Testwerte

- StartDelayMinSeconds = 420
- StartDelayMaxSeconds = 420
- Marker aktiv lassen
- RewardOnDelivery = 1
- Gegnerzahlen niedrig halten

## Empfohlene Livewerte

- PublicCarrierMarkerDelaySeconds passend zur PvP-Balance setzen
- CleanupDroppedCaseAfterSeconds nicht zu niedrig setzen
- RewardItems serverwirtschaftlich pruefen

## Nicht blind aktivieren

- zu viele Gegner
- zu schnelle Restart-Delays
- Markerdelay 0 auf starkem PvP-Server ohne Balancing
