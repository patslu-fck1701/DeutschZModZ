# DeutschZ GroundZero Config-Beschreibung

Mehrstufiges Ground-Zero-Event mit Forschungs-/Containment-Items, Checkpoint-/Extraction-Flow und PvE/PvP-Hotspot-Potenzial.

## Abhaengigkeiten

Abhaengigkeiten: DeutschZ_Core, DeutschZ_ExpansionBridge.

## Profilepfade

- `$profile:DeutschZ/`
- Event-spezifisch: `$profile:DeutschZ/GroundZero/` beziehungsweise vorhandene Mod-Konfigpfade.

## Types.xml

types.xml ist nur als Template mit nominal=0 enthalten. Komponenten sind Missions-/Event-Items und sollten nicht frei auf der Map spawnen.

## Teststatus

Fuer ersten Dedicated-Test vorbereitet. Kein PBO gebaut. Kein lokaler Server-Test durchgefuehrt.
