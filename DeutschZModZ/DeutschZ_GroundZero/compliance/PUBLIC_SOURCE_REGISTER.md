# PUBLIC_SOURCE_REGISTER

Geplante/erlaubte oeffentliche Schnittstellen:

- DayZ Enforce Script API
- DayZ Vanilla Entity/Inventory/Player APIs
- CF API optional
- DayZ Expansion public scripts/wiki optional, nur zur API-, CfgPatches-, requiredAddons-, Klassen- und Signaturpruefung
- DayZ Expansion AI API optional
- DayZ Expansion Marker API optional

Wichtig: Patchnamen fuer Expansion werden nicht geraten. Sie muessen an der installierten Zielkomponente/config.cpp geprueft werden.

Keine Expansion-Klassenkoerper, Funktionen, Assets, Layouts, Kommentare, Header oder Lizenztexte werden uebernommen.
