# DeutschZ OperationDeutschZ

OperationDeutschZ is the DeutschZ campaign and orchestrator mod shell.

This first online-test build is intentionally safe:

- create/load profile folders
- create/load config and state JSON
- print clear boot log lines
- do not trigger live events
- do not change dependencies between the existing event mods

Runtime profile root:

```text
$profile:DeutschZ/OperationDeutschZ/
```
